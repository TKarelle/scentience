# Fonts Directory

## Canela Font

Pour utiliser la police Canela, placez les fichiers de police dans ce dossier avec les noms suivants :

- `Canela-Light.woff2` (ou `.woff`)
- `Canela-Regular.woff2` (ou `.woff`)
- `Canela-Bold.woff2` (ou `.woff`)
- `Canela-LightItalic.woff2` (ou `.woff`)
- `Canela-Italic.woff2` (ou `.woff`)

Une fois les fichiers ajoutés, décommentez les règles `@font-face` dans `src/index.css`.

**Note :** Actuellement, le site utilise Cormorant Garamond comme alternative temporaire (similaire à Canela) via Google Fonts.
